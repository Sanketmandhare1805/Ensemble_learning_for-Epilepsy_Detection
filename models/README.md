# Ensemble Learning for Epileptic Seizure Detection

This repository presents my implementation of an EEG-based epileptic seizure detection framework using Ensemble Learning and Deep Learning techniques.

The implementation was developed independently for learning and educational purposes based on the methodology described in the following paper:

> Mehdi Hosseinzadeh et al., "A Model for Epileptic Seizure Diagnosis Using the Combination of Ensemble Learning and Deep Learning," IEEE Access, 2024.

Unlike the original publication, this repository contains my own implementation and project structure developed to understand the proposed methodology.

## Reference

Mehdi Hosseinzadeh, Parisa Khoshvaght, Samira Sadeghi, et al.

*A Model for Epileptic Seizure Diagnosis Using the Combination of Ensemble Learning and Deep Learning.*

IEEE Access, 2024.

DOI: 10.1109/ACCESS.2024.3457018
